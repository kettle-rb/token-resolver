Overview

Represents plain text content (not a token).

Text nodes hold the literal string content between (or outside of) tokens.
They are frozen after creation.

Examples:

text = Token::Resolver::Node::Text.new("Hello ")
text.to_s     # => "Hello "
text.content  # => "Hello "
